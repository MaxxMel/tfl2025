***Исходное регулярное выражение:***
$((a|b)^*aa^*b(a|b)^*|bb^*a(a|b)^*)abab(a|b)(a|b)$.
Глобально имеет вид $(R_1|R_2)abab(a|b)(a|b)$ 
где $R_1 = R_3aa^*bR_4$, 
$R_3 = R_4 = (a|b)^*$, 
$R_2 = bb^*a(a|b)^*$  - справка для построения - НКА.

## Построение Малого НКА 
Предлагается построить эквивалентное нашему исходному регулярному выражению регулярное выражение, на основе которого построить НКА. 
Дано $R_0 = ((a|b)^* a a^* b (a|b)^* | b b^* a (a|b)^*) abab (a|b)(a|b)$.

Выражение имеет вид $R_0=(A∪B)abab(a∣b)^2$
где $A = (a|b)^* a a^* b (a|b)^*$, $B= b b^* a (a|b)^*$

- Далее по свойству регулярных языков $(X⋅Z)∪(Y⋅Z)=(X∪Y)⋅Z$: $A∪B=((a∣b)^∗aa^∗b∪bb^∗a)(a∣b)^∗$
- Далее $(a∣b)^∗aa^∗b=aa^∗b(a∣b)$
- Мы получаем эквивалентную левую часть: $(aa^∗b∣bb^∗a)(a∣b)^∗$
- Теперь приписываем хвост $abab (a|b)(a|b)$
- Получаем $R_1=(aa^∗b∣bb^∗a)(a∣b)^∗abab(a∣b)(a∣b)$.

Для начала по нему строим $\epsilon$-НКА
